第一大題

(一)首先建立三個用戶，分別為examuser1,examuser2,examuser3密碼為ltlsExam
**解題方式:1.登入時，切換root模式2.開始建立使用者帳號輸入程式碼(useradd examuser1)
再輸入(passwd examuser1)，緊接著會出現一排字(Change password for user examuser1)
就可以輸入密碼，雙重確認，接下來examuser2.examuser3都照著上面步驟作即可完成

(二)請刪除系統中的 examuser3 這個帳號，同時將這個帳號的家目錄與郵件檔案同步刪除
**解題方法--使用-r(連同使用者的家目錄也一起刪除)參數，輸入userdel -r examuser3

(三)examuser1 不小心被管理員刪除了，但是這個帳號的家目錄與相關郵件都還存在。請參考這個帳號可能的家目錄所保留的 UID 與 GID， 並嘗試以該帳號原有的 UID/GID 資訊來重建該帳號。而這個帳號的密碼請給予 ItIsExam 的樣式。
**解題方法:輸入useradd -u (UID程式碼) -U examuser1就會出現如圖(三)所示

第二大題

(一)建立帳戶examuser4
**解題方法:輸入useradd examuser4再輸入passwd examuser4即可輸入密碼，再輸入做確認即可

(二)使用 root 將 /etc/securetty 複製給 examuser4，且這個帳號要能夠完整使用該檔案才行，(即有所有的權限)。
解題方法:首先先切換到root路徑，再來輸入chmod u=rwx,g=rwx,o=rwx /home/examuser4/securetty，再切換cd /home/examuser4,再輸入ll查看權限是否成功

(三)建立一個名為 /examdata/change.txt 的空檔案，這個檔案的擁有者為 sshd，擁有群組為 users，sshd 可讀可寫，users 群組成員可讀， 其他人沒權限。
    且這個檔案的修改日期請調整成 2012 年 12 月 21 日 (日期正確即可，時間隨便)
**解題方法:首先建立一個空檔案，輸入mkdir examdata,在cd進去examdata裡面，touch change.txt，在用ls查看是否成功
再來改變使用者名稱chown sshd change.txt，再來輸入群組名稱chgrp users change.txt，再來改權限輸入chmod u=rw,g=r,o= change.txt，在ll看是否成功

第三大題
(一)請使用 root 的身份建立底下的檔案與權限
**解題方法:輸入cd /dev，enter完後再輸入shm，進入shm夾層建立mkdir unit05，在shm裡面修改unit05權限，輸入chmod u=rwx,g=rwx,o=rx unit05
再來輸入cd unit05裡面建立dir1,dir2,dir3,dir4並修改權限(在unit05 夾層輸入mkdir dir1,再來chmod u=rwx,g=rx,o=r dir1)(在unit05 夾層輸入mkdir dir2,再來chmod u=rwx,g=rx,o=x dir2)
(在unit05 夾層輸入mkdir dir3,再來chmod u=rwx,g=rx,o=rx dir3)(在unit05 夾層輸入mkdir dir4,再來chmod u=rwx,g=rwx,o=rwx dir4)再輸入ll看是否成功
再來建立dir1-4 的file (輸入cp /etc/hosts dev/shm/unit05/dir1/file1 進去dir1 再來改權限chmod u=rw,g=r,o=r file1在cd ..回去unit05)(輸入cp /etc/hosts dev/shm/unit05/dir2/file2 進去dir2 再來改權限chmod u=rw,g=r,o=r file2在cd ..回去unit05 )
(輸入cp /etc/hosts dev/shm/unit05/dir3/file3 進去dir3 再來改權限chmod u=rw,g=rw,o=rw file3在cd ..回去unit05 )(輸入cp /etc/hosts dev/shm/unit05/dir4/file4 進去dir4 再來改權限chmod u=rw,g=,o= file4在cd ..回去unit05 )
